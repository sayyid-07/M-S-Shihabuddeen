Namma Vastra 🧵 - AI Powered Handloom Marketplace & Trend Guide

Namma Vastra is a modern Android application built with Jetpack Compose designed to bridge the gap between traditional handloom weavers and global consumers. By integrating AI-driven trend forecasts and a direct-to-consumer marketplace, the app empowers authentic artisans while celebrating rich cultural heritage.

 Key Features

* 🛍️ Direct-to-Consumer Marketplace:Safe and secure platform for authentic weavers to showcase and sell handloom products directly without middlemen.
* 🔮 AI Trend Guide: Uses Google's Gemini API to analyze current fashion datasets and suggest high-demand color patterns, modern motifs, and trending cuts to traditional artisans.
* 📸 Smart Fabric Verification: AI-driven camera scanning to help buyers detect authentic weave counts and differentiate pure handloom from powerloom fabric.
* 📍 Artisan Story Maps: Interactive regional maps highlighting where specific garments (like Khadi, Ikat, or Kanjeevaram) are woven, fostering community-driven heritage tourism.
* 💬 Multilingual Support: Voice-to-text and multi-language user interfaces ensuring ease of access for rural weavers across various regions.



🛠️ Tech Stack & Architecture

* Frontend: Kotlin, Jetpack Compose, Material 3 Design System
* Architecture: MVVM (Model-View-ViewModel) + Clean Architecture
* Asynchronous Operations: Kotlin Coroutines & Flow
* Backend: Firebase (Authentication, Firestore Database, Cloud Storage)
* AI Integration: Google GenAI SDK (Gemini Pro Vision / Flash)
